Cette classe permet de définir les opérations sur les nombres complexes. Effectivement, les nombres complexes n'étant pas gérés en Java, cette classe permet de définir une implémentation des fonctions de bases de calculs que l'on peut effectuer sur les nombres complexes, définies dans la classe NombreComplexe. Ces opérations sont :

  • addition()
  • soustraction()
  • etc
